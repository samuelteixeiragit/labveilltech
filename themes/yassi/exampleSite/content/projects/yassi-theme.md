Yassi is a modern Hugo theme designed for developers, writers, and creators who want a clean, focused platform for their content.

## Features

- Timeline-style blog layout with chronological flow
- Dark mode with persistent user preferences
- Flexible project showcase (carousel or grid)
- Word cloud taxonomy displays
- Responsive design optimized for all devices
- Social media integration
- Clean typography and readability focus

## Technology

Built with Hugo's templating system and modern CSS, Yassi is fast, lightweight, and easy to customize.

## Open Source

Yassi is open source and available on GitHub. Contributions are welcome!
